JShelpers

some helper functions, bundled in window.fl.helpers

  • replaceDotByComma(s): replaces dots with commas (e.g. "45.2%" => "45,2%")
    param: String/Number (typecasted to String)
    returns: String
  • getPercentageStrFromNumber(n): converts number to percantage string (e.g. 0.45 => "45%")
    param: Number
    returns: String
  • inheritObject(childObj, prototypeObj): compatibility-workaround for Object.create;
    params: childObj (Obj to inherit), prototypeObj (Obj to inherit from)
    returns: /
    Usage: fl.helpers.inheritObject(concreteClass, abstractClass);
  • localStorageEnabled(): checks if localStorge is available
  • getIndexOfObjectInArrayByPropertyvalue(array, attribute, value): get index of an object in an array
    params: array (array to be searched in), attribute (attribute we should check to get you the result), value (the value the attribute should have)
    e.g. you're looking for an object { myAttribute : "myValue" }, then get desired object as follows: array[f(array, "myAttribute", "myValue")];
    returns: Integer (-1 if nothing found)
  • isFunction(functionObject): checks if given object is a function
